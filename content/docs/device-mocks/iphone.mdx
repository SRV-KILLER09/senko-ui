## Installation
```bash
npx shadcn@latest add https://senko-ui.vercel.app/r/iphone-mockup.json
```

## Usage
```tsx
import IPhoneMockup from "@/registry/iphone-mockup"
```
```tsx
<IPhoneMockup>
  <div className="p-4">Your content here</div>
</IPhoneMockup>
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `children` | `ReactNode` | `—` | Content rendered inside the iPhone screen |
| `screenClassName` | `string` | `—` | Additional classes for the screen area |
| `scale` | `number` | `1` | Scale factor for resizing the mockup |
| `className` | `string` | `—` | Additional classes for the outer wrapper |
