## Installation
```bash
npx shadcn@latest add your-registry-url/android-mockup
```

## Usage
```tsx
import AndroidMockup from "@/components/android-mockup"
```
```tsx
<AndroidMockup />
```

## With Content
```tsx
<AndroidMockup>
  </div>
</AndroidMockup>
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `children` | `ReactNode` | `—` | Content rendered inside the screen |
| `screenClassName` | `string` | `—` | Additional classes for the screen area |
| `scale` | `number` | `1` | Scale the mockup up or down |
| `className` | `string` | `—` | Additional classes for the outer wrapper |
