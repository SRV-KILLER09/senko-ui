# TS-UI

**TS-UI** is a signature collection of copy-paste components designed for the premium web. Built with **Next.js**, **Tailwind CSS**, and **Framer Motion**, these components prioritize minimalist aesthetics and high-performance motion.

## Philosophy

Most component libraries are built for utility — TS-UI is built for **identity**. We provide the signature elements that make your SaaS feel expensive, stable, and forward-thinking.

---

## Core Features

- **Native Framer Motion** — 60fps interactions designed for fluidity and responsiveness
- **Tailwind-First** — Fully themeable using standard CSS variables and utility classes
- **Radix UI Optimized** — Built on accessible primitives with full keyboard and screen-reader support
- **Type-Safe** — Developed with TypeScript for a robust developer experience

---

## Quick Start

Add components using our CLI, which integrates directly with the `shadcn/ui` ecosystem:
```bash
npx shadcn@latest add https://ts-ui.com/registry/component-name.json
```

For manual installation, copy the component code into your `components/` directory and install dependencies:
```bash
npm install framer-motion lucide-react clsx tailwind-merge
```
