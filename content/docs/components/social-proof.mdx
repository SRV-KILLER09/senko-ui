## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `heading` | `ReactNode` | `"Trusted by 120,000+ founders..."` | Custom heading text or JSX |
| `avatars` | `{ src, alt, name }[]` | 5 default avatars | Array of avatar objects |
| `rating` | `number` | `5` | Star rating out of 5 |
| `className` | `string` | `—` | Additional classes |
