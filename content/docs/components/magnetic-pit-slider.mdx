## Installation

```bash
npx shadcn@latest add https://senko-ui-nine.vercel.app/r/magnetic-pit-slider.json
```

## Usage

```tsx
import { MagneticPitSlider } from "@/components/magnetic-pit-slider"
```

```tsx
<MagneticPitSlider defaultValue={50} onChange={(val) => console.log(val)} />
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `value` | `number` | `—` | Controlled value |
| `defaultValue` | `number` | `50` | Initial uncontrolled value |
| `min` | `number` | `0` | Minimum value |
| `max` | `number` | `100` | Maximum value |
| `step` | `number` | `1` | Step increment |
| `barCount` | `number` | `48` | Number of equalizer bars |
| `onChange` | `(value: number) => void` | `—` | Change handler |
| `className` | `string` | `—` | Additional classes |
