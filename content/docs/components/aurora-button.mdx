## Installation
```bash
npx shadcn@latest add your-registry-url/aurora-button
```

## Usage
```tsx
import AuroraButton from "@/components/aurora-button"
```
```tsx
<AuroraButton label="Get Started" />
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `label` | `string` | `"Click me"` | Button text |
| `beamColor` | `string` | `"139, 92, 246"` | Beam color as `"r, g, b"` |
| `duration` | `string` | `"3s"` | Beam rotation speed |
| `onClick` | `() => void` | `—` | Click handler |
| `className` | `string` | `—` | Additional classes |
