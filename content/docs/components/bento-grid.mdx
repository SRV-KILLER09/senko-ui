## Usage
```tsx
import { BentoGrid, BentoCard } from "@/components/bento"
import { Search } from "lucide-react"
```
```tsx
<BentoGrid>
  <BentoCard
    className="md:col-span-3 md:row-span-1"
    name="Tagging"
    description="Neural auto-organization for your files."
    background={
      <div className="absolute inset-0 flex items-center justify-center">
        {/* your background visual */}
      </div>
    }
    Icon={Search}
    href="/docs"
    cta="Explore"
    glowColor="139, 92, 246"
  />
</BentoGrid>
```

## Layout

The grid uses `grid-cols-12` by default. Control each card's size using `col-span` and `row-span`:
```tsx
{/* Big card вЂ” takes 7 cols, 2 rows */}
<BentoCard className="md:col-span-7 md:row-span-2" ... />

{/* Small card вЂ” takes 5 cols, 1 row */}
<BentoCard className="md:col-span-5 md:row-span-1" ... />

{/* Full width */}
<BentoCard className="md:col-span-12" ... />
```

## Props

### BentoGrid

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `children` | `ReactNode` | `вЂ”` | BentoCard components |
| `className` | `string` | `вЂ”` | Override grid columns, rows, gap |

### BentoCard

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `name` | `string` | `вЂ”` | Card heading |
| `description` | `string` | `вЂ”` | Card subtext |
| `background` | `ReactNode` | `вЂ”` | Visual rendered behind the content |
| `Icon` | `React.ElementType` | `вЂ”` | Icon shown in the top left |
| `href` | `string` | `вЂ”` | Link destination for the CTA |
| `cta` | `string` | `вЂ”` | CTA button label |
| `glowColor` | `string` | `"255,255,255"` | Cursor glow color as `"r, g, b"` |
| `className` | `string` | `вЂ”` | Controls col/row span and any overrides |
