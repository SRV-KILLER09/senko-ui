## Installation
```bash
npx shadcn@latest add your-registry-url/pill-navbar
```

## Usage
```tsx
import PillNavbar from "@/components/GlassyTopBar"

<PillNavbar
  logo={<span className="font-bold">TS.UI</span>}
  items={[
    { label: "Components", href: "/docs" },
    { label: "GitHub", href: "https://github.com" },
  ]}
  actions={[<button>Get Started</button>]}
/>
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `items` | `NavItem[]` | `—` | Navigation links |
| `logo` | `ReactNode` | `—` | Logo element |
| `actions` | `ReactNode[]` | `—` | Action buttons on the right |
| `fixed` | `boolean` | `true` | Whether navbar is fixed to viewport |
| `className` | `string` | `—` | Additional classes |

### NavItem

| Prop | Type | Description |
| --- | --- | --- |
| `label` | `string` | Link text |
| `href` | `string` | Link URL |
