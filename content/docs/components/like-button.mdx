## Installation

```bash
npx shadcn@latest add https://senko-ui.vercel.app/r/like-button.json
```

## Usage

```tsx
import { LikeButton } from "@/components/like-button"
```

```tsx
<LikeButton initialCount={128} />
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `initialCount` | `number` | `0` | Starting like count |
| `defaultLiked` | `boolean` | `false` | Whether the button starts in liked state |
| `className` | `string` | `—` | Additional CSS classes |
