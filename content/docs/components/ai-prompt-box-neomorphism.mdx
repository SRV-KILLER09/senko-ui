## Features

- **Neomorphism Design** — Soft extruded shadow style with inset active states
- **Image Uploads** — Click, drag-and-drop, or paste images from clipboard
- **Voice Recording** — Full recording UI with animated waveform visualizer
- **Mode Toggles** — Search (blue), Think (purple), Canvas (orange) with expanding labels
- **Inset Active States** — Buttons press in with `neo-shadow-inset` on activation
- **Loading State** — Shows stop button while streaming

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `onSend` | `(message: string, files?: File[]) => void` | `—` | Called on submit |
| `isLoading` | `boolean` | `false` | Shows stop button when true |
| `placeholder` | `string` | `"Ask me anything..."` | Textarea placeholder |
| `className` | `string` | `—` | Additional classes |
