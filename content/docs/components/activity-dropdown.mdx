## Features

- **Collapsible Panel** — Click to expand/collapse with smooth spring animations
- **Tabs** — Switch between All and Unread notifications
- **Mark as Read** — Click individual items or "Mark all read"
- **Animated Entries** — Items animate in with staggered delays
- **Unread Badge** — Live pulsing indicator on the bell icon
