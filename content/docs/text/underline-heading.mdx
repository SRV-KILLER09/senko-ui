## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `content` | `string[]` | `—` | Lines of text to render |
| `highlightContent` | `string[]` | `—` | Words or phrases to apply the highlight to |
| `className` | `string` | `—` | Additional classes for the heading |
| `highlightClassName` | `string` | `—` | Classes for the highlight bar — controls color, height, rotation |
| `textGradientClass` | `string` | `—` | Classes for custom text color or gradient |
| `behind` | `boolean` | `false` | Renders the highlight behind the text instead of over it |
