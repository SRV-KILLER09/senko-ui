## Installation
```bash
npx shadcn@latest add https://senko-ui.vercel.app/r/wavy-grid-background.json
```

## Usage
```tsx
import WavyGridBackground from "@/components/wavy-grid"
```
```tsx
<div className="relative w-full h-[400px]">
  <WavyGridBackground mode="contained" />
  <div className="relative z-10">
    Your content here
  </div>
</div>
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `squareSize` | `number` | `4` | Size of each grid square in pixels |
| `gridGap` | `number` | `2` | Gap between grid squares in pixels |
| `maxOpacity` | `number` | `0.25` | Maximum opacity of the grid squares |
| `height` | `number` | `400` | Height in pixels, only applies in `fixed` mode |
| `mode` | `"fixed" \| "contained"` | `"fixed"` | `fixed` pins to viewport top, `contained` fills parent |
