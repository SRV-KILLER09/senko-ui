## Features

- **Light Beams** — 12 animated beams travel horizontally and vertically along grid lines
- **Pulsing Nodes** — 15 nodes pulse at grid intersections with staggered delays
- **Glow Patches** — 5 large blurred radial patches drift across the grid for ambient depth
- **Edge Fade** — Grid fades to transparent at edges with a radial mask
- **Vignette** — Subtle vignette overlay adds cinematic depth
- **SSR Safe** — Randomized elements are generated client-side to avoid hydration mismatches

## CSS Variables

Add these to your `globals.css` to match your theme:

```css
:root {
  --grid-line: rgba(0, 0, 0, 0.08);
  --grid-dot: rgba(0, 0, 0, 0.15);
  --grid-beam: rgba(99, 102, 241, 0.6);
  --grid-patch: rgba(99, 102, 241, 0.05);
  --vignette: rgba(0, 0, 0, 0.15);
}

.dark {
  --grid-line: rgba(255, 255, 255, 0.06);
  --grid-dot: rgba(255, 255, 255, 0.12);
  --grid-beam: rgba(129, 140, 248, 0.5);
  --grid-patch: rgba(129, 140, 248, 0.04);
  --vignette: rgba(0, 0, 0, 0.4);
}
```
